The problem

Models are forced to remember things they were never built to remember.

Vector search is fuzzy

Embedding-based retrieval approximates relevance. It cannot guarantee that the right fact comes back.

Prompt stuffing is expensive

Dumping history into the context window burns tokens and degrades attention over long sessions.

State drifts silently

Without versioning, a character’s memory becomes a black box. You cannot diff, branch, or rollback.

The solution

Memory as infrastructure.

GUAC treats memory as a first-class system component: queryable, versioned, and compressible.

🌿

Core Knowledge Graph

Stable facts live as structured YAML. Relationships are deterministic, not approximate.

🌳

Git-Native Memory

Every turn becomes a commit. Branch conversations, inspect diffs, and rollback state.

🧠

Natural Memory Layer

Messages are scored on importance, novelty, recency, and repetition. Only salient memories survive.

📦

Filesystem Addressing

Query memory by path: /memory/projects/firebac/warfare. No semantic search required.

🗜️

Temporal Compression

Old conversations are recursively summarized, giving AI effectively infinite memory.

⚡

Groq Reasoning Layer

Ultra-low-latency inference turns retrieved context into responses in milliseconds.

FAQ

Questions, answered.

What makes GUAC different from a vector database?

Vector databases approximate relevance with embeddings. GUAC uses structured knowledge graphs and addressable conversation history for deterministic retrieval.

Can I self-host GUAC?

Yes. The core CLI is MIT-licensed and runs entirely on your machine. Commercial tiers add cloud sync, team collaboration, and enterprise controls.

Does GUAC require Groq?

No. Groq is the default for low-latency inference, but the CLI falls back to a local echo mode and the future server will support multiple inference providers.

How does temporal compression work?

Old messages are recursively summarized into higher-level summaries. Individual low-score messages are pruned, while key facts and narratives are preserved.
