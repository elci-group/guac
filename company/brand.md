# GUAC Brand Book

## Identity

**Name:** GUAC  
**Pronunciation:** /ɡwæk/ — like the dip.  
**Stands for:** Git + Groq Augmented Cognition

GUAC is memorable, short, and a little playful. It turns a dry technical acronym into something human. The avocado metaphor runs deep: layers, a stable core, and the idea that good memory systems should feel organic, not synthetic.

---

## Mission

> Give every AI system a persistent, structured, version-controlled memory.

## Vision

> The operating system for thought: a cognitive substrate where models reason over memory instead of manufacturing it.

---

## Positioning

**For** engineering teams and product builders who ship AI characters, agents, and assistants.  
**Who need** deterministic, long-term memory they can inspect, version, and compress.  
**GUAC is** a Git-native knowledge graph and conversation memory system.  
**Unlike** vector databases and prompt-stuffing hacks,  
**GUAC treats memory as infrastructure** — structured, addressable, and auditable.

## Voice & Tone

**Voice:** Direct, technical, slightly irreverent.

- **Do:** Use concrete examples, short sentences, and active verbs.
- **Do:** Explain trade-offs honestly.
- **Don’t:** Use AI hype words like "revolutionary," "magical," or "unprecedented."
- **Don’t:** Pretend memory is solved. It’s a hard problem and we’re building the best tool for it.

**Tone by context:**

| Context | Tone |
|---------|------|
| Website | Confident, problem-solution, a little bold. |
| Docs | Plain, instructive, example-heavy. |
| Social | Witty, concise, meme-aware but never mean. |
| Enterprise | Measured, security-aware, ROI-focused. |

## Logo Concept

The GUAC mark combines three ideas:

1. **Avocado pit** — the stable core of memory.
2. **Git branch** — forking, merging, versioning.
3. **Neural node** — cognition and connection.

It is a rounded, symmetrical shape: a central circle (the pit/core) with two curved branches extending upward and outward. The wordmark uses **JetBrains Mono Bold** in all caps: `GUAC`.

**Usage rules:**
- Always give the mark clear space equal to the height of the "U".
- Use the dark version on light backgrounds, light version on dark.
- Do not stretch, rotate, or add effects.

See `website/assets/logo.svg` for the master file.

## Messaging Framework

### Problem
AI models forget. Vector search is fuzzy. Prompt stuffing is expensive and opaque.

### Solution
GUAC gives AI systems structured, Git-backed memory with deterministic retrieval, branching, and compression.

### Proof
- Every conversation turn is a commit.
- Knowledge graph lookups are exact, not approximate.
- Memory is scored, compressed, and addressable like a filesystem.
- Latency target: <50ms end-to-end.

### CTA
- **Developers:** "Install the CLI and give your agent a memory."
- **Teams:** "Book a memory architecture review."
- **Researchers:** "Read the memory-compression paper."

## Brand Guardrails

1. **Always uppercase** GUAC in text. Never "Guac" or "guac" unless in a casual social context.
2. **Never claim AGI.** We build memory infrastructure.
3. **Be specific about latency.** Use "target" or "measured" prefixes.
4. **Credit Git and Groq** where relevant. We stand on their shoulders.
5. **Open-source friendly.** The core CLI is MIT. Commercial layers are additive.
